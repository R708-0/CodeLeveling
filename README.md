#### Description:

Este proyecto es una mezcla de una aplicacin de tareas, un portfolio y un sistema de juego para aprender programacion diseñado para ayudar a personas que están aprendiendo programación a mantener la motivación, la disciplina y el enfoque a largo plazo. A diferencia de un portfolio tradicional, que solo muestra proyectos finalizados, esta plataforma convierte el proceso de aprendizaje en un sistema de progreso continuo, donde cada acción cuenta y cada pequeño avance se traduce en crecimiento visible.

El usuario no solo presenta lo que sabe hacer, sino que también ve de manera clara qué habilidades está desarrollando, qué tareas debe completar y cómo cada esfuerzo impacta directamente en su evolución como desarrollador.
